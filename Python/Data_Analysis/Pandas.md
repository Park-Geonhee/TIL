# Pandas 

## Pandas 사용하기
- Pandas : 2차원 정렬 데이터를 다루는 모듈입니다.
- 데이터 분석에서 `Pandas`의  `Series`, `DataFrame` 을 주로 다룹니다.
```python
import pandas as pd
from pandas import Series, DataFrame
```

## Series & DataFrame

### Series
- 1차원의 자료 구조이며, pandas의 기본 단위입니다.
- 하나의 데이터 타입을 허용합니다.

### DataFrame
- 2차원의 자료 구조입니다.
